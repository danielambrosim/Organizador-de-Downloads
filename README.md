# 📁 Organizador de Downloads

Script Python que organiza arquivos por extensão em subpastas.

## Como usar

```bash
# Organizar pasta Downloads
python organizer.py

# Organizar pasta específica
python organizer.py --folder /caminho/da/pasta

# Simular (apenas mostra o que seria feito)
python organizer.py --folder ./Downloads --dry-run

# Ver ajuda
python organizer.py --help
